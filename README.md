## Getting Started

Follow the instructions to set up the project on your local machine.

1. Clone the repo

```sh
git clone https://github.com/{your-github-username}/ddance.git
```

2. Install NPM packages

```sh
npm install
```

3. Start react
    ```sh
    npm start
    ```
